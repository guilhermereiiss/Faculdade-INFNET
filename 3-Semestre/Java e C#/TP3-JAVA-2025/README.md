# Java - Conceitos de Classe, Objeto, Campos e Métodos

![Java Logo](https://upload.wikimedia.org/wikipedia/en/3/30/Java_programming_language_logo.svg)

## Descrição

Este repositório contém uma série de exercícios práticos para o aprendizado de conceitos fundamentais da Programação Orientada a Objetos (POO) em Java. Os exercícios abordam desde a criação de classes e objetos até o uso de getters, setters e métodos de manipulação de dados.

## Exercícios

### Exercício 1 - Conceitos de Classe, Objeto, Campos e Métodos
Explicação teórica e exemplo prático ilustrando os conceitos fundamentais da POO.

### Exercício 2 - Criando a Classe "Produto"
Criação de uma classe que representa um produto de supermercado com atributos essenciais como nome, preço e quantidade em estoque.

### Exercício 3 - Métodos Básicos da Classe "Produto"
Implementação de métodos para alteração de preço e quantidade, além de um método para exibição das informações do produto.

### Exercício 4 - Testando a Classe "Produto"
Criação de um método `main` para instanciar objetos da classe `Produto`, modificar seus atributos e exibir os resultados.

### Exercício 5 - Criando Métodos de Propriedade (Getters e Setters)
Implementação de getters e setters para acessar e modificar os atributos de forma segura.

### Exercício 6 - Adicionando Construtores à Classe "Produto"
Criação de um construtor para facilitar a inicialização de objetos da classe `Produto`.

### Exercício 7 - Modelando uma Conta Bancária
Definição da classe `Conta`, contendo atributos essenciais como titular, número, agência, saldo e data de abertura.

### Exercício 8 - Criando Métodos para a Classe "Conta"
Implementação de métodos para saque, depósito e cálculo de rendimento da conta bancária.

### Exercício 9 - Testando a Classe "Conta"
Criação de um programa de teste para validar o funcionamento da classe `Conta`.

### Exercício 10 - Definindo Classes para Formas Geométricas
Criação das classes `Circulo` e `Esfera`, contendo atributos essenciais para representação de figuras geométricas.

### Exercício 11 - Criando Métodos de Cálculo
Adicionação de métodos para calcular área e volume das formas geométricas criadas no exercício anterior.

## Requisitos
- Java Development Kit (JDK) 8 ou superior.
- Um editor de código ou IDE, como IntelliJ IDEA, Eclipse ou VS Code.

## Como Executar
1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/seu-repositorio.git
   ```
2. Acesse o diretório do projeto:
   ```bash
   cd seu-repositorio
   ```
3. Compile os arquivos Java:
   ```bash
   javac NomeDaClasse.java
   ```
4. Execute o programa:
   ```bash
   java NomeDaClasse
   ```
